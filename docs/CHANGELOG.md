# Changelog

## 0.2.0

- Added GitHub Action support with single-comment post/update behavior.
- Added OpenAI and Anthropic provider adapters.
- Added `revix init` and `revix skill init <reviewer-id>`.
- Added `skills.paths` loading for external reviewer skill packs.
- Added v0.2 user documentation and release packaging.

## 0.1.0

- Added deterministic local review pipeline.
- Added built-in reviewer skills, PR classification, conflict detection,
  synthesis, final composition, CLI review/check commands, and evaluation tools.
